# 냠냠레벨업 지원 안내

## 지원 범위
- 학교 검색이 되지 않는 경우
- 급식 정보가 표시되지 않는 경우
- 체험 모드와 실제 학교 모드 구분
- 사진 권한 안내
- 부모 연결 초대 코드 안내
- 데이터 삭제 방법

## 자주 묻는 질문

### 급식 정보가 없다고 나와요.
방학, 재량휴업일, 급식 미운영일이거나 NEIS에 아직 식단이 등록되지 않았을 수 있습니다. 실제 학교 선택 상태에서는 샘플 급식으로 대체하지 않습니다.

### 샘플 급식은 언제 나오나요?
사용자가 체험 모드를 직접 선택한 경우에만 샘플 급식이 표시됩니다.

### 알레르기 메뉴는 먹어도 되나요?
앱은 알레르기 안전을 보장하지 않습니다. 선택한 알레르기와 관련된 메뉴는 한 입 도전보다 보호자와 학교 안내 확인이 먼저입니다.

### 사진은 어디에 저장되나요?
급식판 사진은 기기 내부에만 저장됩니다. 부모 모드에는 먹은 정도, 한 입 도전 기록, 알레르기 주의만 공유되고 사진 원본은 올라가지 않습니다.

### 아이가 급식 결과를 올리면 부모가 알림을 받을 수 있나요?
부모 모드에서 알림을 허용하고 아이 초대 코드로 연결하면, 아이가 공유 기록을 올렸을 때 급식 결과 알림을 받을 수 있습니다.

### 회원가입이 필요한가요?
필요 없습니다. 별명과 학교만 선택하면 사용할 수 있습니다.

## 문의 전 확인할 항목
- 학교 이름을 정확히 입력했는지 확인
- 인터넷 연결 확인
- 앱 설정의 학교 다시 선택 사용
- 사진 권한 설정 확인
- 데이터 삭제 전 되돌릴 수 없다는 점 확인
