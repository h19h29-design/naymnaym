## 설명
냠냠레벨업은 학교 급식 정보를 바탕으로 아이가 어려워하는 반찬을 기록하고, 작은 한 입 도전을 캐릭터 성장으로 연결하는 무료 iPhone 앱입니다.

회원가입 없이 별명과 학교만 등록해 시작할 수 있습니다. 실제 NEIS 학교기본정보와 급식식단정보를 조회해 오늘 급식과 월간 급식을 보여주고, 안 먹기 쉬운 반찬은 먹은 정도와 어려운 이유를 남길 수 있습니다.

초등학생은 쉽고 귀여운 화면으로, 중학생은 미션과 EXP 중심의 게임형 화면으로, 고등학생은 차분한 자기관리형 화면으로 사용할 수 있습니다. 부모 모드에서는 여러 아이를 초대 코드로 연결해 아이별 한 입 도전, 알레르기 주의, 선택 공유된 급식판 사진을 확인할 수 있습니다.

알레르기 정보는 안전을 보장하지 않습니다. 선택한 알레르기와 관련된 메뉴는 한 입 도전보다 보호자와 학교 안내 확인을 먼저 하도록 표시합니다.

샘플 급식은 체험 모드를 사용자가 직접 선택한 경우에만 표시됩니다. 실제 학교 조회 실패, API 키 없음, 급식 없는 날에는 샘플로 대체하지 않고 정확한 상태를 안내합니다.

## 심사 제출 메모
냠냠레벨업은 NEIS 공공데이터 기반 무료 급식 식습관 코칭 앱입니다. 회원가입 없이 별명과 학교를 선택해 사용합니다. 샘플 데이터는 사용자가 체험 모드를 직접 선택한 경우에만 표시되며, 실제 학교 급식 조회 실패 시 샘플로 대체하지 않습니다.

알레르기 안내는 안전을 보장하지 않고 학교 안내와 보호자 판단이 우선임을 앱 안에 명시했습니다. 사진 기능은 급식판 기록용이며 친구 얼굴, 이름표, 반/번호가 나오지 않도록 안내합니다. 기록 공유와 사진 공유는 별도 선택이며, 사진은 기본적으로 기기 내부에 저장되고 부모 공유를 켠 사진만 부모 모드 공유 대상입니다.

## App Privacy 응답 초안
- 추적: 아니요
- 제3자 광고: 아니요
- 광고 식별자 사용: 아니요
- 회원가입 계정 정보 수집: 아니요
- 정확한 위치정보 수집: 아니요
- 연락처 정보 수집: 아니요
- 사용자 콘텐츠: 먹은 정도, 한 입 도전, 어려운 이유, 공유 선택한 급식판 사진은 기본적으로 기기 내부 저장이며, 부모 공유를 켠 항목만 CloudKit 공유 대상. 앱 안에서 기록 공유와 사진 공유는 별도 선택
- 건강 정보: 알레르기/주의 선택값은 선택 입력이며 앱 기능 제공과 안전 안내 목적으로만 사용. 의학 진단, 추적, 광고 목적으로 사용하지 않음
- 식별자: 부모 연결용 초대 코드와 `childLinkId`는 다자녀 기록 분리와 공유 연결에만 사용
- 기타 데이터: 학교 코드와 날짜는 NEIS 공공데이터 API 조회에 사용될 수 있음
- Privacy Manifest: UserDefaults required reason API, 선택 부모 공유용 사용자 콘텐츠/사진/건강/사용자 ID 데이터 타입, 추적 없음 선언

## 참고한 Apple 제출 기준
- App Store Connect App Privacy는 개인정보 처리방침 URL을 요구한다.
- App Privacy 정보는 App Store Connect에서 제출 전 입력해야 한다.
- App Review Guidelines는 앱 안에서 개인정보 처리방침 접근이 쉬워야 한다고 안내한다.
- Required reason API 사용 항목은 Privacy Manifest에 사유를 선언해야 한다.
