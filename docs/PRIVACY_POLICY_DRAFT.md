# 냠냠레벨업 개인정보 처리방침 초안

시행일: 2026-06-20

냠냠레벨업은 학교 급식 정보를 활용해 아이의 한 입 도전과 식습관 기록을 돕는 무료 iPhone 앱입니다. 이 문서는 앱이 어떤 정보를 어떻게 사용하는지 설명합니다.

## 1. 회원가입
냠냠레벨업은 회원가입을 요구하지 않습니다. 이름, 이메일, 전화번호, 계정 비밀번호를 수집하지 않습니다.

## 2. 앱 안에 저장되는 정보
다음 정보는 기본적으로 사용자의 기기 내부에 저장됩니다.

- 별명
- 선택한 학교
- 선택한 알레르기 코드
- 먹은 정도 기록
- 한 입 도전 기록
- 어려운 이유 기록
- 캐릭터 레벨과 뱃지
- 급식판 사진과 사진 메타데이터
- 부모 연결 설정

## 3. 공공데이터 조회
앱은 급식 조회를 위해 NEIS 학교기본정보 API와 NEIS 급식식단정보 API를 사용합니다. 선택한 학교 코드와 조회 날짜가 NEIS 공공데이터 API 요청에 포함될 수 있습니다.

## 4. 사진
급식판 사진은 사용자의 기기 내부에만 저장됩니다. 사진 원본은 부모 모드나 서버 부모 동기화 대상으로 업로드하지 않습니다. 앱은 친구 얼굴, 이름표, 반/번호가 포함된 사진 촬영을 권장하지 않습니다.

## 5. 부모 연결
부모 연결은 Supabase 기반 서버를 사용합니다. 공유 대상은 사용자가 선택한 먹은 정도, 한 입 도전 기록, 알레르기 주의로 제한됩니다. 부모 공유를 켠 기록은 부모 모드 표시 기능을 위해 서버에 저장될 수 있습니다. 부모는 연결된 아이의 학교 코드로 NEIS 급식식단정보 API를 다시 조회해 오늘 급식 메뉴를 볼 수 있습니다. 부모가 알림을 허용하면 아이가 급식 결과를 올렸을 때 부모 기기로 알림을 보낼 수 있습니다. 급식판 사진은 서버 부모 동기화에 포함하지 않습니다.

## 6. 수집하지 않는 정보
냠냠레벨업은 다음 정보를 수집하지 않습니다.

- 이름
- 이메일
- 전화번호
- 정확한 위치정보
- 연락처
- 광고 식별자
- 결제 정보
- 공개 프로필
- 자유 채팅 내용

## 7. 광고와 추적
냠냠레벨업은 광고가 없고 인앱결제가 없습니다. 광고 추적 또는 제3자 광고 네트워크를 사용하지 않습니다.

## 8. 알레르기와 영양 안내
알레르기 정보와 영양 안내는 교육용 참고 정보입니다. 앱은 알레르기 안전, 건강 상태, 영양 상태를 진단하거나 보장하지 않습니다. 학교 안내와 보호자 판단이 항상 우선입니다.

## 9. 데이터 삭제
앱의 설정 > 데이터 관리에서 도전 기록, 프로필, 전체 데이터를 삭제할 수 있습니다. 기기 내부 저장 데이터는 삭제 후 되돌릴 수 없습니다.

## 10. 문의
문의와 지원 요청은 지원 페이지의 안내에 따라 접수합니다.
