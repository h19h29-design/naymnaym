냠냠레벨업으로 돌아가기

지원 안내

냠냠레벨업 사용 중 학교 검색, 급식 조회, 사진 기록, 부모 연결 문제가 있을 때 아래 항목을 먼저 확인해 주세요.

급식 정보가 없다고 나와요

방학, 재량휴업일, 급식 미운영일이거나 NEIS에 아직 식단이 등록되지 않았을 수 있습니다. 실제 학교 선택 상태에서는 샘플 급식으로 대체하지 않습니다.

API 키 또는 학교 설정 오류가 나와요

급식 정보를 불러오지 못했다는 안내가 나오면 API 키, 학교 선택, 네트워크 상태를 확인해 주세요. 앱은 이 상태에서 샘플 급식을 대신 보여주지 않습니다.

샘플 급식은 언제 나오나요?

사용자가 체험 모드를 직접 선택한 경우에만 샘플 급식이 표시됩니다.

알레르기 메뉴는 먹어도 되나요?

앱은 알레르기 안전을 보장하지 않습니다. 선택한 알레르기와 관련된 메뉴는 한 입 도전보다 보호자와 학교 안내 확인이 먼저입니다.

사진은 어디에 저장되나요?

급식판 사진은 기본적으로 기기 내부에 저장됩니다. 부모 공유를 켠 사진만 부모 모드 공유 대상입니다.

부모 연결이 안 돼요

아이폰에서 보호자 연결 초대 코드를 먼저 생성하고, 부모 모드의 아이 추가 화면에 같은 코드를 입력해 주세요. iCloud 사용 가능 상태와 네트워크 연결도 확인해 주세요.

문의 전 확인

  • 학교 이름을 정확히 입력했는지 확인
  • 인터넷 연결 확인
  • 앱 설정의 학교 다시 선택 사용
  • 사진 권한 설정 확인
  • 데이터 삭제 전 되돌릴 수 없다는 점 확인

문의 채널

앱 오류나 급식 조회 문제는 냠냠레벨업 GitHub Issues에 남길 수 있습니다. 공개 게시판이므로 아이 이름, 학교 상세 정보, 알레르기 개인 정보, 사진, 연락처는 올리지 마세요.
