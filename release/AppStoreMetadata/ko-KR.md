## 설명

냠냠레벨업은 학교 급식 정보를 바탕으로 아이가 어려워하는 반찬을 기록하고, 한 입 도전과 캐릭터 성장으로 식습관 변화를 돕는 무료 iPhone 앱입니다.

단순한 급식표 앱이 아니라, 아이가 스스로 작은 도전을 기록하고 부모가 부드럽게 응원할 수 있도록 설계되었습니다.

주요 기능:

- NEIS 공공데이터 기반 학교 검색과 급식식단정보 조회
- 오늘 급식과 월간 급식 확인
- 안 먹기, 냄새 맡기, 한 입 도전, 반 정도 먹기, 잘 먹은 기록
- 기록 XP, 도전 XP, 균형 XP, 안전 XP 기반 레벨업
- 초등학생, 중학생, 고등학생, 부모 모드
- 모드별 테마와 캐릭터 성장
- 알레르기 주의 메뉴 한 입 도전 잠금
- 급식판 사진 기록과 선택적 부모 공유
- 부모 모드 다자녀 요약과 칭찬 카드
- SNS 공유 카드와 iOS 기본 공유 시트
- 회원가입 없이 시작
- 광고와 인앱결제 없음

샘플 급식은 사용자가 체험 모드를 직접 선택한 경우에만 표시됩니다. 실제 학교 급식 조회 실패, API 키 없음, 급식 없는 날에는 샘플로 대체하지 않고 정확한 안내를 보여줍니다.

알레르기 정보와 영양 안내는 교육용 참고 정보입니다. 앱은 알레르기 안전, 건강 상태, 영양 상태를 진단하거나 보장하지 않으며 학교 안내와 보호자 판단이 항상 우선입니다.

## 새 기능

- 첫 실행 인트로와 캐릭터 애니메이션 개선
- 실제 급식, 급식 없음, API 오류, 체험 모드 상태 분리
- 한 입 도전과 먹은 정도 기록 기반 XP 시스템 개선
- 알레르기 주의 메뉴 안전 XP 처리
- 부모 다자녀 요약과 선택 사진 공유
- SNS 공유 카드와 이미지 저장 기능
- 개인정보 안내와 지원 문서 정리

## 심사용 설명

냠냠레벨업은 NEIS 공공데이터 기반 무료 급식 식습관 코칭 앱입니다.

- 회원가입이 없으므로 테스트 계정은 필요하지 않습니다.
- 첫 실행에서 학교 등록 또는 체험 모드로 앱을 확인할 수 있습니다.
- 체험 모드는 실제 학교 급식이 아니라 샘플 데이터임을 명확히 표시합니다.
- 실제 학교 급식 조회 실패, API 키 없음, 급식 없는 날에는 샘플 급식을 자동 표시하지 않습니다.
- 사진 기능은 급식판 기록용이며 기본 로컬 저장입니다.
- 부모 공유는 사용자가 기록 공유와 사진 공유를 선택한 경우만 사용합니다.
- 알레르기 정보는 참고용이며 학교 안내와 보호자 판단이 우선임을 앱 안에 표시합니다.
- 광고, 인앱결제, 분석 SDK, 추적 SDK를 사용하지 않습니다.

## App Privacy 답변 초안

정확한 입력은 `release/AppStoreMetadata/app-privacy-draft.md`를 기준으로 App Store Connect에서 최종 확인합니다.

- Tracking: 없음
- Third-party advertising: 없음
- Analytics SDK: 없음
- Contact Info: 수집 안 함
- Location: 수집 안 함
- User Content: 사진/기록은 기본 로컬 저장, 부모 공유 선택 시 CloudKit 전송 가능
- Health & Fitness: 알레르기 선택값과 식사 기록은 건강 관련 정보로 해석될 수 있어 검토 필요
- Identifiers: 부모 연결용 초대 코드와 `childLinkId`가 앱 기능 제공 목적으로 사용됨
- Diagnostics: 앱 자체 진단 SDK 없음. Apple 기본 진단은 App Store Connect 정책에 따름
